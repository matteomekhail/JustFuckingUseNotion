# Just Fucking Use Notion

A straightforward landing page that helps people overcome productivity app paralysis by recommending Notion as a simple, effective solution.

**Live site:** [justfuckingusenotion.com](https://justfuckingusenotion.com)

## About

This project is part of the [justfuckinguse*](https://github.com/Kuberwastaken/awesome-fucking-websites) family of websites. It provides a humorous but practical perspective on the common "which productivity app should I use?" dilemma.

The site covers:
- Comparison with Apple Notes
- Comparison with Obsidian
- Why Notion works for most use cases
- Overview of the free plan

## Tech Stack

- [Astro](https://astro.build/)
- Vanilla CSS with Inter font
- Zero client-side JavaScript

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
